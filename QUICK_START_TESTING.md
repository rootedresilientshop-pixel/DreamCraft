## The 4-Week Plan

### Week 1: Testing Starts
- Recruit 15-20 testers
- Testers complete 45-minute test plan
- Collect bugs and feedback
- Daily triage and quick fixes

### Week 2: Bug Fixes
- Fix all critical/high bugs
- Deploy to staging
- Analyze feedback themes
- Plan improvements

### Week 3: Verification
- Second round testing (on fixes)
- Final production checks
- Prepare for launch

### Week 4: Launch
- Deploy to production (Feb 7)
- Public announcement
- Monitor metrics

## What Testers Do (45 minutes)

### For Creators (~20 mins)
1. Create account as Creator
2. Create idea using template
3. Find and invite collaborators
4. Browse marketplace

### For Collaborators (~20 mins)
1. Create account as Collaborator
2. Browse and search ideas
3. Evaluate ideas (timing test)
4. Propose collaboration

### Feedback (~5 mins)
- Report bugs using template
- Fill feedback form
- Note what works/doesn't work
